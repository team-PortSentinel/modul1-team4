============================================================
프로젝트명 : 취약점 찾아 조
============================================================

[프로젝트 개요]
본 프로젝트는 Nmap 스캔 결과를 기반으로 최신 취약점(CVE/CVSS)을 자동 분석하고,
머신러닝(Random Forest) 모델을 활용하여 위험도를 예측하는
AI 기반 침투 테스트 지원 시스템입니다.

사용자는 Nmap 결과만 입력하면 서비스 정보, 관련 취약점,
위험도, 대응 방안 등을 한 번에 확인할 수 있습니다.

------------------------------------------------------------
[주요 기능]
1. Nmap 스캔 결과 분석
2. 포트 및 서비스 정보 추출
3. 최신 CVE/CVSS 검색 (Web Search, NVD, Vulners API)
4. AI 기반 취약점 설명 및 대응 방안 제공
5. Random Forest 기반 위험도 예측
6. Streamlit Dashboard 결과 시각화
7. PDF 보고서 생성

------------------------------------------------------------
[프로젝트 구조]
scanner.py           : Nmap 스캔 수행
parser.py            : 스캔 결과 JSON 변환 및 정제
openai_analyzer.py   : AI 취약점 분석 및 Web Search 연동
risk_predictor.py    : 위험도 예측 모델
dashboard.py         : Streamlit UI 및 시각화

------------------------------------------------------------
[실행 순서]
1. 대상 IP 입력
2. Nmap 스캔 수행
3. 스캔 결과 파싱
4. 최신 취약점 검색
5. 위험도 예측
6. AI 분석 결과 생성
7. Dashboard 출력 및 보고서 생성

------------------------------------------------------------
[사용 기술]
- Python 3.x
- Nmap / python-nmap
- OpenAI API
- NVD API / Vulners API
- Scikit-learn (Random Forest)
- Streamlit

------------------------------------------------------------
[주의 사항]
- 본 프로젝트는 취약점 분석 및 대응 우선순위 추천을 목적으로 합니다.
- 실제 공격 코드 제공이나 불법 침투 행위는 포함하지 않습니다.
- 테스트는 허가된 시스템 또는 공개 테스트 환경에서만 수행해야 합니다.
